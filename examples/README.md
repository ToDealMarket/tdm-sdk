## TDM SDK Examples

These examples are intentionally small and direct.

- `authorize-basic.ts`: call the payment gate directly
- `checkout-basic.ts`: create a hosted checkout session
- `framework-mode-basic.ts`: protect one web route with `chargeFetchHandler(...)`
- `gateway-clients-basic.ts`: share one transport across several friendly clients
- `session-tanks-basic.ts`: create a sub-tank and delegate a session
- `make-payable-route.ts`: wrap an application function with `makePayable(...)`

They are reference snippets for the current public surface, not a full starter app.
